Bikers spot is an online bike showroom management system that provides solution for show rooms for organizing the data using web application. This application helps administrator and customer to update data and retreive data in step by step process while selling nd buying a vehicle. Using this system management the administration team can view customer information, booking details, drive request details and add data of a vehicle , model...etc to the system. Here in this system it is also developed for maintaining the show room activities like , customer maintenance, customer quotation generation, bike booking details, loan EMI calculator, test drive request , customer feedbacks , new user registration...etc. The system is efficient in generating reports which will help in the maintainance of the showroom easily. Bike showroom management system describes the complete process of selling a bike to a customer from the dealer?s showroom. Before selling , the bike belongs to a fixed asset of the dealer?s showroom. So the main point of this sceanrio is posting the Bike from a fixed asset to a current asset. While executing the process, the dealer can manually maintain the bike business transaction type, which means, the Bike can be set as a new or used bike to sell to the customer.
